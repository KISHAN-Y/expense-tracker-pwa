## 🐛 Troubleshooting

### Common Issues

**"API not working"**
- Check Google Apps Script deployed
- Verify API endpoint in config.js
- See DEPLOYMENT.md

**"Offline mode not working"**
- Check Service Worker status
- Clear browser cache
- See TESTING.md

**"Won't install"**
- Ensure HTTPS (or localhost)
- Check manifest.json
- See QUICKSTART.md
